Cliente Controller

Gestión de clientes del banco

GET /api/clientes

Obtener todos los clientes

Recupera la lista completa de clientes registrados

Respuesta

Lista de todos los clientes registrados

GET /api/clientes/{id}

Obtener un cliente por ID

Recupera los datos de un cliente específico por su ID

Parámetros

id - ID del cliente

Respuesta

GET /api/clientes/dui/{dui}

Obtener un cliente por DUI

Recupera los datos de un cliente específico por su número de DUI

Parámetros

dui - Número de DUI del cliente

Respuesta

Datos del cliente con el DUI especificado

Crear un nuevo cliente

Registra un nuevo cliente en el sistema

Body

Datos del cliente a crear

Respuesta

PUT /api/clientes/{id}

Actualizar un cliente

Parámetros

id - ID del cliente a actualizar

Body

Datos actualizados del cliente

Respuesta

Datos del cliente actualizado

DELETE /api/clientes/{id}

Eliminar un cliente

Elimina un cliente del sistema

Parámetros

Obtener un movimiento por ID

GET /api/movimientos/cuenta/{idCuenta}

Parámetros

idCuenta - ID de la cuenta

GET /api/movimientos/fechas

Obtener movimientos por fechas

Recupera todos los movimientos entre fechas específicas

Parámetros

fechaFin - Fecha de fin (yyyy-MM-dd)

Respuesta

Lista de movimientos entre las fechas especificadas

GET /api/movimientos/tipo/{tipoMovimiento}

Obtener movimientos por tipo

Recupera todos los movimientos de un tipo específico

Parámetros

tipoMovimiento - Tipo de movimiento (DEPOSITO, RETIRO, TRANSFER)

Respuesta

Lista de movimientos del tipo especificado

Respuesta
